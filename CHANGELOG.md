# Changelog

## ✨ Added

- Beautiful animated modals
- Confirm dialogs
- Toast notifications
- SVG icons system
- Progress bar for notifications
- Glassmorphism UI
- Input support (`text`, `textarea`, `select`)

- Smooth modal animations
- SweetAlert2-like SVG animations

- Redesigned notification icons
- Smaller and cleaner toast icons
- Better modal animations
- Improved info icon
- Better spacing and shadows
- Enhanced UI consistency

## 🛠 Fixed

- Modal close animation

- Overlay blur rendering
- Icon alignment
